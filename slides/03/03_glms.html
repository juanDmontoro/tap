Modelos paramétricos de respuesta no continua

Problemas de clasificación

 

  • Problemas de clasificación: respuesta cualitativa/categórica

  • Técnicas de clasificación

  • Conceptos aplicables de los problemas de regresión: aprendizaje supervisado, partición de datos, sobreajuste,subajuste, etc.

  • ¿Regresión lineal en problemas de clasificación?

    • Si la respuesta es binaria \(X\hat{\beta}\) es un estimador de \(P(Y=1|X)\)
    • Pero las predicciones \(P(Y|X)\) del modelo lineal no necesariamente estarán dentro del intervalo \([0,1]\)
    • Imposibilidad de acomodar respuestas cualitativas cuando tenemos más de dos clases (la respuesta debería tener un ordenamiento implícito, e incluso entonces la diferencia entre niveles de la respuesta sería arbitraria)

Tipos de respuesta y modelos paramétricos adecuados

 

  • La respuesta es continua: regresión lineal, Ridge, Lasso
  • La respuesta es binaria: regresión logística (también probit)
  • La respuesta es categórica (más de dos opciones): regresión multinomial
  • La respuesta es un proceso de conteo (0,1,2…): regresión Poisson (o binomial negativa si sobredispersión)

Los modelos de regresión lineal, logístico, multinomial o Poisson (entre otros) se engloban dentro del llamado modelo lineal generalizado

Modelos lineales generalizados (GLMs)

Componentes de un GLM

 

  1. Un componente estocástico que especifica la distribución de probabilidad de la variable respuesta (familia EDM, p. ej., normal, binomial, Poisson).

\[y_i\sim EDM(\mu_i,\sigma^2)\]

  1. Un componente sistemático o predictor lineal:
    \[\nu_i = \beta_1 + \beta_2 x_{2i} + \ldots + \beta_p x_{pi}\]

  2. Función de enlace, es decir, una forma funcional que conecta la media de la respuesta con el componente sistemático (el predictor lineal):
    \[g(\mu_i) = \nu_i\]

Dado que los GLM asumen una distribución de probabilidad específica para las respuestas de la familia EDM, se pueden usar procedimientos numéricos de estimación por máxima verosimilitud (p.e. descenso del gradiente estocástico) para estimar los parámetros:
\[-l(\beta \mid x, y)\] es la función de pérdida a minimizar.

El modelo lineal

 

  1. Distribución gaussiana para la respuesta \(y_i\):
    \[y_i \sim N(\mu_i, \sigma^2)\]

  2. Predictor lineal:
    \[\nu_i = \beta_0+ \beta_1 x_{i} + \beta_2 x_{2i} + \ldots + \beta_p x_{pi}\]

  3. Función de enlace identidad:
    \[\nu_i = \mu_i\]

El modelo lineal estima la media condicional de la variable respuesta en función del predictor lineal:

\[E[y \mid X] = \beta_0 + \beta_1 x_{1i}+ \beta_2 x_{2i} + \ldots + \beta_p x_{pi}\]

Respuesta binaria: el modelo logístico

Sea un resultado binario \(y \in \{0,1\}\) y \(x_1, x_2, \ldots, x_p\) las variables explicativas. En este caso:

  1. El componente estocástico (distribución Bernoulli):
    \[P(Y=y_i;\mu_i) = \mu_i^{y_i}(1-\mu_i)^{(1-y_i)}\]

  2. El predictor lineal:
    \[\nu_i = \beta_0 + \beta_1 x_{1i} + \beta_2 x_{2i} + \ldots + \beta_p x_{pi}\]

  3. La función de enlace:
    \[\nu_i = g(\mu_i) = \log\left(\frac{\mu_i}{1-\mu_i}\right)= \log\left(\frac{P(y=1|X)}{P(y=0|X)}\right)\]

Combinando estos elementos, el modelo logit predice la media condicional de la respuesta como:

\[P(y_i=1 \mid X) = \mu_i = \frac{\exp(X\beta)}{1+\exp(X\beta)}\]

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

def _sigmoid(z):
    # Estable para extremos numéricos
    z = np.clip(z, -500, 500)
    return 1.0 / (1.0 + np.exp(-z))

def plot_logistic_partial_curve(coefs, intercept, X, var, y,
                                fixed='mean', fixed_values=None,
                                n_points=200, show_scatter=True, ax=None):
    """
    Dibuja la curva de probabilidad predicha de un modelo logístico en función de 'var',
    manteniendo el resto de predictores fijos.

    Parámetros
    ----------
    coefs : array-like, shape (p,)
        Coeficientes del modelo en el mismo orden que X.columns.
    intercept : float
        Intercepto (a).
    X : pd.DataFrame
        Datos con exactamente las columnas de predictores del modelo.
    var : str
        Nombre de la variable a representar en el eje X (ej.: 'var0').
    fixed : {'mean', 'median'} o float, opcional (por defecto 'mean')
        Cómo fijar el resto de variables:
        - 'mean'   -> usa la media de X para cada variable.
        - 'median' -> usa la mediana de X.
        - float    -> usa ese valor para todas las variables (excepto 'var').
        Ignorado si se especifica `fixed_values`.
    fixed_values : dict opcional
        Permite fijar algunas (o todas) las otras variables a valores concretos.
        Ej.: {'var1': 0.0, 'var2': 10}
    n_points : int, opcional
        Número de puntos en la malla para la curva.
    show_scatter : bool, opcional
        Si True, dibuja también el scatter de prob. predichas en los puntos observados.
    ax : matplotlib.axes.Axes, opcional
        Un eje existente sobre el que dibujar.

    Devuelve
    --------
    ax : matplotlib.axes.Axes
        El eje con la figura.
    """
    # Validaciones básicas
    coefs = np.asarray(coefs).ravel()
    assert isinstance(X, pd.DataFrame), "X debe ser un DataFrame de pandas."
    assert var in X.columns, f"'{var}' no está en X.columns."
    assert len(coefs) == len(X.columns), "coefs y columnas de X no coinciden en longitud."

    cols = X.columns
    idx_var = list(cols).index(var)

    # Vector base de valores fijos
    if fixed_values is not None:
        # base: medias (o medianas) para lo no especificado
        if fixed == 'median':
            x_fixed = X.median(numeric_only=True)
        else:
            x_fixed = X.mean(numeric_only=True)
        for k, v in fixed_values.items():
            if k in cols:
                x_fixed[k] = float(v)
    else:
        if fixed == 'median':
            x_fixed = X.median(numeric_only=True)
        elif fixed == 'mean':
            x_fixed = X.mean(numeric_only=True)
        elif np.isscalar(fixed):
            x_fixed = pd.Series({c: float(fixed) for c in cols})
        else:
            raise ValueError("Parámetro 'fixed' debe ser 'mean', 'median' o un escalar.")

    # Malla sobre la variable de interés
    x_grid = np.linspace(X[var].min(), X[var].max(), n_points)

    # Parte constante: intercepto + sum_j!=var coefs[j] * x_fixed[j]
    mask_other = np.ones(len(cols), dtype=bool)
    mask_other[idx_var] = False
    const = float(intercept) + float(np.dot(coefs[mask_other], x_fixed[mask_other].values))

    # Predicción parcial variando solo 'var'
    z_curve = const + coefs[idx_var] * x_grid
    p_curve = _sigmoid(z_curve)

    # Preparar eje
    created_ax = False
    if ax is None:
        fig, ax = plt.subplots()
        created_ax = True

    # Scatter de probabilidades predichas en puntos observados (opcional)
    if show_scatter:
        z_obs = intercept + X.values @ coefs
        p_obs = _sigmoid(z_obs)
        ax.scatter(X[var].values, p_obs, s=20, alpha=0.35, c=y, cmap='bwr',
                   label='Prob. predicha (observaciones)',  edgecolors='none')

    # Curva parcial (teórica) con otros predictores fijos
    ax.plot(x_grid, p_curve, color='crimson', lw=2,
            label=f'Curva parcial: {var} | otros fijos')

    # Etiquetas y formato
    ax.set_xlabel(var)
    ax.set_ylabel('Probabilidad de fumador (predicción)')
    ax.set_title(f'Modelo logístico: probabilidad')
    ax.set_ylim(-0.02, 1.02)
    ax.grid(True, alpha=0.25)
    ax.axhline(y=0, color='gray', linestyle='--')
    ax.axhline(y=1, color='gray', linestyle='--')
    ax.axhline(y=0.5, color='green', linestyle='-.')
    ax.legend()

    if created_ax:
        plt.xlabel('Coste del seguro')
        plt.show()

    return ax


# Curva parcial respecto a 'charges', fijando el resto a su media
plot_logistic_partial_curve(coefs=coefs, intercept=a, X=X_test,y=y_test, var='charges', fixed='mean')

Regresión multiclase: el modelo logístico multinomial

 

Si la respuesta \(y\) tiene \(k > 2\) clases, usamos un modelo multinomial (también un GLM). Sus componentes:

  1. Componente estocástico: la respuesta sigue un modelo multinomial, es decir, puede tomar uno de \(K\) valores. La probabilidad de que haya \(y_1, y_2, \ldots, y_K\) resultados en cada categoría \(k =1 \ldots, K\) de la respuesta \(y\) en \(n\) ensayos está dada por:
    \[ P(y_1, y_2, \ldots, y_K) = \frac{n!}{y_1! y_2! \cdots y_K!} p_1^{y_1} p_2^{y_2} \cdots p_K^{y_K} \]

  2. Un predictor lineal para cada categoría \(j\)-ésima (donde \(j = 1, 2, \ldots, K\)):
    \[ \nu_j = \beta_{0j} + \beta_{1j} x_1 + \beta_{2j} x_2 + \cdots + \beta_{pj} x_p \]

  3. Una función de enlace soft-max (en sklearn). Para la categoría \(j\)-ésima:
    \[ \log \left(\frac{P(Y = j)}{\sum_k P(Y=k)}\right) = \nu_j \] donde \(P(Y = j)\) es la probabilidad de la categoría \(j\) y \(\sum_k P(Y=k)\) es la suma de las probabilidades para todas las categorias (funciona como normalización).

Regresión logística multinomial

 

La media condicional está dada por:

\[P(y=j|X)=\frac{\exp{(X\beta_j)}}{\sum_k \exp{(X\beta_k)}}\]

  • Conceptualmente esto supone estimar \(K\) modelos logit binarios (habitual en aprendizaje automático).

No obstante:

  • Determinadas implementaciones elijen una categoría de referencia (o base) y estiman solo \(k-1\) modelos logit (habitual en determinados paquetes econométricos/estadísticos). En este caso se establece \(\beta=0\) para una categoría base o de referencia.

  • En el caso anterior la elección de la categoría base es arbitraria (aunque algunos pueden preferir la clase más frecuente o basar la elección en consideraciones teóricas).

  • La función de enlace al usar la categoría de referencia \(k\) se convierte en: \[ \log \left(\frac{P(Y = j)}{P(Y = k)}\right) = \nu_j\]

Ejemplo: modelo multinomial

Modelo de Poisson

Los datos de conteo pueden modelarse usando las especificaciones Poisson (media igual a varianza) y Binomial Negativa (sobredispersión). En el primer caso:

  1. Componente estocástico: la respuesta sigue un modelo Poisson,
    \[ P(y_i|\lambda) = \frac{\exp({-\lambda})\, \lambda^{y_i}}{y_i!} \]

  2. El predictor lineal es:
    \[ \nu_j = \beta_{0} + \beta_{1} x_1 + \beta_{2} x_2 + \cdots + \beta_{p} x_p \]

  3. Una función de enlace logarítmica que asegura que los conteos sean siempre no negativos:
    \[ \log (\lambda) = X \beta \]

Observa que esto implica:
\[E[y|X]=\lambda=\exp({X\beta})\]

Ejemplo: modelo de Poisson

Resumen de GLMs

 

Modelo Distribución (Componente estocástico) Predictor lineal Función de enlace
Lineal \(y_i \sim N(\mu_i,\sigma^2)\) \(\nu_i = \beta_0 + \beta_1 x_1 + \dots + \beta_p x_p\) Identidad: \(\, \nu_i = \mu_i\)
Logístico (binario) \(y_i \sim \text{Bernoulli}(\mu_i)\) \(\nu_i = \beta_0 + \beta_1 x_1 + \dots + \beta_p x_p\) Logit: \(\, \nu_i = \log\left(\dfrac{\mu_i}{1-\mu_i}\right)\)
Logístico multinomial \(y_i \sim \text{Multinomial}(p_1,\dots,p_k)\) \(\nu_j = \beta_{0j} + \beta_{1j} x_1 + \dots + \beta_{pj} x_p\) (para \(j=1,\dots,K\)) Logit: \(\, \log\left(\dfrac{P(Y=j)}{P(Y=k)}\right)=\nu_j\)
Poisson (conteos) \(y_i \sim \text{Poisson}(\lambda)\) \(\nu_i = \beta_0 + \beta_1 x_1 + \dots + \beta_p x_p\) Log: \(\, \log(\lambda)=X\beta\)

 

Otros GLM incluyen respuestas continuas no negativas o tiempos de supervivencia.

Regularización en el modelo lineal generalizado

GLMs regularizados

 

  • La regularización se utiliza para evitar el sobreajuste al entrenar GLM con fines predictivos.

  • Usamos el mismo enfoque que con el modelo lineal: se introduce un término de regularización en la función de pérdida que penaliza la magnitud de los coeficientes estimados.

  • Hay un parámetro de ajuste \(\lambda\) que controla la cantidad de regularización.

  • El valor óptimo de \(\lambda\) se elige mediante remuestreo (validación cruzada).

La función de pérdida de un GLM regularizado incluye un regularizador \(R\):
\[-l(\beta|x,y)+\lambda R(\beta)\]

Tipos de penalización

 

Como en el caso del modelo lineal, la penalización puede ser \(L_1\), \(L_2\) o una combinación de ambas:

  1. LASSO:
    \[R(\beta)=||\beta||_1=\sum_j|\beta_j|\]
  2. Ridge:
    \[R(\beta)=||\beta||_2=\sum_j\beta_j^2\]
  3. Elastic net:
    \[R(\beta)=\alpha ||\beta||_1+ \frac{1}{2}(1-\alpha)||\beta||_2\]
