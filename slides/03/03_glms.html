Modelos param√©tricos de respuesta no continua

Problemas de clasificaci√≥n

 

  • Problemas de clasificaci√≥n: respuesta cualitativa/categ√≥rica

  • T√©cnicas de clasificaci√≥n

  • Conceptos aplicables de los problemas de regresi√≥n: aprendizaje supervisado, partici√≥n de datos, sobreajuste,subajuste, etc.

  • ¬øRegresi√≥n lineal en problemas de clasificaci√≥n?

    • Si la respuesta es binaria \(X\hat{\beta}\) es un estimador de \(P(Y=1|X)\)
    • Pero las predicciones \(P(Y|X)\) del modelo lineal no necesariamente estar√°n dentro del intervalo \([0,1]\)
    • Imposibilidad de acomodar respuestas cualitativas cuando tenemos m√°s de dos clases (la respuesta deber√≠a tener un ordenamiento impl√≠cito, e incluso entonces la diferencia entre niveles de la respuesta ser√≠a arbitraria)

Tipos de respuesta y modelos param√©tricos adecuados

 

  • La respuesta es continua: regresi√≥n lineal, Ridge, Lasso
  • La respuesta es binaria: regresi√≥n log√≠stica (tambi√©n probit)
  • La respuesta es categ√≥rica (m√°s de dos opciones): regresi√≥n multinomial
  • La respuesta es un proceso de conteo (0,1,2‚Ä¶): regresi√≥n Poisson (o binomial negativa si sobredispersi√≥n)

Los modelos de regresi√≥n lineal, log√≠stico, multinomial o Poisson (entre otros) se engloban dentro del llamado modelo lineal generalizado

Modelos lineales generalizados (GLMs)

Componentes de un GLM

 

  1. Un componente estoc√°stico que especifica la distribuci√≥n de probabilidad de la variable respuesta (familia EDM, p. ej., normal, binomial, Poisson).

\[y_i\sim EDM(\mu_i,\sigma^2)\]

  1. Un componente sistem√°tico o predictor lineal:
    \[\nu_i = \beta_1 + \beta_2 x_{2i} + \ldots + \beta_p x_{pi}\]

  2. Funci√≥n de enlace, es decir, una forma funcional que conecta la media de la respuesta con el componente sistem√°tico (el predictor lineal):
    \[g(\mu_i) = \nu_i\]

Dado que los GLM asumen una distribuci√≥n de probabilidad espec√≠fica para las respuestas de la familia EDM, se pueden usar procedimientos num√©ricos de estimaci√≥n por m√°xima verosimilitud (p.e. descenso del gradiente estoc√°stico) para estimar los par√°metros:
\[-l(\beta \mid x, y)\] es la funci√≥n de p√©rdida a minimizar.

El modelo lineal

 

  1. Distribuci√≥n gaussiana para la respuesta \(y_i\):
    \[y_i \sim N(\mu_i, \sigma^2)\]

  2. Predictor lineal:
    \[\nu_i = \beta_0+ \beta_1 x_{i} + \beta_2 x_{2i} + \ldots + \beta_p x_{pi}\]

  3. Funci√≥n de enlace identidad:
    \[\nu_i = \mu_i\]

El modelo lineal estima la media condicional de la variable respuesta en funci√≥n del predictor lineal:

\[E[y \mid X] = \beta_0 + \beta_1 x_{1i}+ \beta_2 x_{2i} + \ldots + \beta_p x_{pi}\]

Respuesta binaria: el modelo log√≠stico

Sea un resultado binario \(y \in \{0,1\}\) y \(x_1, x_2, \ldots, x_p\) las variables explicativas. En este caso:

  1. El componente estoc√°stico (distribuci√≥n Bernoulli):
    \[P(Y=y_i;\mu_i) = \mu_i^{y_i}(1-\mu_i)^{(1-y_i)}\]

  2. El predictor lineal:
    \[\nu_i = \beta_0 + \beta_1 x_{1i} + \beta_2 x_{2i} + \ldots + \beta_p x_{pi}\]

  3. La funci√≥n de enlace:
    \[\nu_i = g(\mu_i) = \log\left(\frac{\mu_i}{1-\mu_i}\right)= \log\left(\frac{P(y=1|X)}{P(y=0|X)}\right)\]

Combinando estos elementos, el modelo logit predice la media condicional de la respuesta como:

\[P(y_i=1 \mid X) = \mu_i = \frac{\exp(X\beta)}{1+\exp(X\beta)}\]

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

def _sigmoid(z):
    # Estable para extremos num√©ricos
    z = np.clip(z, -500, 500)
    return 1.0 / (1.0 + np.exp(-z))

def plot_logistic_partial_curve(coefs, intercept, X, var, y,
                                fixed='mean', fixed_values=None,
                                n_points=200, show_scatter=True, ax=None):
    """
    Dibuja la curva de probabilidad predicha de un modelo log√≠stico en funci√≥n de 'var',
    manteniendo el resto de predictores fijos.

    Par√°metros
    ----------
    coefs : array-like, shape (p,)
        Coeficientes del modelo en el mismo orden que X.columns.
    intercept : float
        Intercepto (a).
    X : pd.DataFrame
        Datos con exactamente las columnas de predictores del modelo.
    var : str
        Nombre de la variable a representar en el eje X (ej.: 'var0').
    fixed : {'mean', 'median'} o float, opcional (por defecto 'mean')
        C√≥mo fijar el resto de variables:
        - 'mean'   -> usa la media de X para cada variable.
        - 'median' -> usa la mediana de X.
        - float    -> usa ese valor para todas las variables (excepto 'var').
        Ignorado si se especifica `fixed_values`.
    fixed_values : dict opcional
        Permite fijar algunas (o todas) las otras variables a valores concretos.
        Ej.: {'var1': 0.0, 'var2': 10}
    n_points : int, opcional
        N√∫mero de puntos en la malla para la curva.
    show_scatter : bool, opcional
        Si True, dibuja tambi√©n el scatter de prob. predichas en los puntos observados.
    ax : matplotlib.axes.Axes, opcional
        Un eje existente sobre el que dibujar.

    Devuelve
    --------
    ax : matplotlib.axes.Axes
        El eje con la figura.
    """
    # Validaciones b√°sicas
    coefs = np.asarray(coefs).ravel()
    assert isinstance(X, pd.DataFrame), "X debe ser un DataFrame de pandas."
    assert var in X.columns, f"'{var}' no est√° en X.columns."
    assert len(coefs) == len(X.columns), "coefs y columnas de X no coinciden en longitud."

    cols = X.columns
    idx_var = list(cols).index(var)

    # Vector base de valores fijos
    if fixed_values is not None:
        # base: medias (o medianas) para lo no especificado
        if fixed == 'median':
            x_fixed = X.median(numeric_only=True)
        else:
            x_fixed = X.mean(numeric_only=True)
        for k, v in fixed_values.items():
            if k in cols:
                x_fixed[k] = float(v)
    else:
        if fixed == 'median':
            x_fixed = X.median(numeric_only=True)
        elif fixed == 'mean':
            x_fixed = X.mean(numeric_only=True)
        elif np.isscalar(fixed):
            x_fixed = pd.Series({c: float(fixed) for c in cols})
        else:
            raise ValueError("Par√°metro 'fixed' debe ser 'mean', 'median' o un escalar.")

    # Malla sobre la variable de inter√©s
    x_grid = np.linspace(X[var].min(), X[var].max(), n_points)

    # Parte constante: intercepto + sum_j!=var coefs[j] * x_fixed[j]
    mask_other = np.ones(len(cols), dtype=bool)
    mask_other[idx_var] = False
    const = float(intercept) + float(np.dot(coefs[mask_other], x_fixed[mask_other].values))

    # Predicci√≥n parcial variando solo 'var'
    z_curve = const + coefs[idx_var] * x_grid
    p_curve = _sigmoid(z_curve)

    # Preparar eje
    created_ax = False
    if ax is None:
        fig, ax = plt.subplots()
        created_ax = True

    # Scatter de probabilidades predichas en puntos observados (opcional)
    if show_scatter:
        z_obs = intercept + X.values @ coefs
        p_obs = _sigmoid(z_obs)
        ax.scatter(X[var].values, p_obs, s=20, alpha=0.35, c=y, cmap='bwr',
                   label='Prob. predicha (observaciones)',  edgecolors='none')

    # Curva parcial (te√≥rica) con otros predictores fijos
    ax.plot(x_grid, p_curve, color='crimson', lw=2,
            label=f'Curva parcial: {var} | otros fijos')

    # Etiquetas y formato
    ax.set_xlabel(var)
    ax.set_ylabel('Probabilidad de fumador (predicci√≥n)')
    ax.set_title(f'Modelo log√≠stico: probabilidad')
    ax.set_ylim(-0.02, 1.02)
    ax.grid(True, alpha=0.25)
    ax.axhline(y=0, color='gray', linestyle='--')
    ax.axhline(y=1, color='gray', linestyle='--')
    ax.axhline(y=0.5, color='green', linestyle='-.')
    ax.legend()

    if created_ax:
        plt.xlabel('Coste del seguro')
        plt.show()

    return ax


# Curva parcial respecto a 'charges', fijando el resto a su media
plot_logistic_partial_curve(coefs=coefs, intercept=a, X=X_test,y=y_test, var='charges', fixed='mean')

Regresi√≥n multiclase: el modelo log√≠stico multinomial

 

Si la respuesta \(y\) tiene \(k > 2\) clases, usamos un modelo multinomial (tambi√©n un GLM). Sus componentes:

  1. Componente estoc√°stico: la respuesta sigue un modelo multinomial, es decir, puede tomar uno de \(K\) valores. La probabilidad de que haya \(y_1, y_2, \ldots, y_K\) resultados en cada categor√≠a \(k =1 \ldots, K\) de la respuesta \(y\) en \(n\) ensayos est√° dada por:
    \[ P(y_1, y_2, \ldots, y_K) = \frac{n!}{y_1! y_2! \cdots y_K!} p_1^{y_1} p_2^{y_2} \cdots p_K^{y_K} \]

  2. Un predictor lineal para cada categor√≠a \(j\)-√©sima (donde \(j = 1, 2, \ldots, K\)):
    \[ \nu_j = \beta_{0j} + \beta_{1j} x_1 + \beta_{2j} x_2 + \cdots + \beta_{pj} x_p \]

  3. Una funci√≥n de enlace soft-max (en sklearn). Para la categor√≠a \(j\)-√©sima:
    \[ \log \left(\frac{P(Y = j)}{\sum_k P(Y=k)}\right) = \nu_j \] donde \(P(Y = j)\) es la probabilidad de la categor√≠a \(j\) y \(\sum_k P(Y=k)\) es la suma de las probabilidades para todas las categorias (funciona como normalizaci√≥n).

Regresi√≥n log√≠stica multinomial

 

La media condicional est√° dada por:

\[P(y=j|X)=\frac{\exp{(X\beta_j)}}{\sum_k \exp{(X\beta_k)}}\]

  • Conceptualmente esto supone estimar \(K\) modelos logit binarios (habitual en aprendizaje autom√°tico).

No obstante:

  • Determinadas implementaciones elijen una categor√≠a de referencia (o base) y estiman solo \(k-1\) modelos logit (habitual en determinados paquetes econom√©tricos/estad√≠sticos). En este caso se establece \(\beta=0\) para una categor√≠a base o de referencia.

  • En el caso anterior la elecci√≥n de la categor√≠a base es arbitraria (aunque algunos pueden preferir la clase m√°s frecuente o basar la elecci√≥n en consideraciones te√≥ricas).

  • La funci√≥n de enlace al usar la categor√≠a de referencia \(k\) se convierte en: \[ \log \left(\frac{P(Y = j)}{P(Y = k)}\right) = \nu_j\]

Modelo de Poisson

Los datos de conteo pueden modelarse usando las especificaciones Poisson (media igual a varianza) y Binomial Negativa (sobredispersi√≥n). En el primer caso:

  1. Componente estoc√°stico: la respuesta sigue un modelo Poisson,
    \[ P(y_i|\lambda) = \frac{\exp({-\lambda})\, \lambda^{y_i}}{y_i!} \]

  2. El predictor lineal es:
    \[ \nu_j = \beta_{0} + \beta_{1} x_1 + \beta_{2} x_2 + \cdots + \beta_{p} x_p \]

  3. Una funci√≥n de enlace logar√≠tmica que asegura que los conteos sean siempre no negativos:
    \[ \log (\lambda) = X \beta \]

Observa que esto implica:
\[E[y|X]=\lambda=\exp({X\beta})\]

Ejemplo: modelo de Poisson

Resumen de GLMs

 

Modelo Distribuci√≥n (Componente estoc√°stico) Predictor lineal Funci√≥n de enlace
Lineal \(y_i \sim N(\mu_i,\sigma^2)\) \(\nu_i = \beta_0 + \beta_1 x_1 + \dots + \beta_p x_p\) Identidad: \(\, \nu_i = \mu_i\)
Log√≠stico (binario) \(y_i \sim \text{Bernoulli}(\mu_i)\) \(\nu_i = \beta_0 + \beta_1 x_1 + \dots + \beta_p x_p\) Logit: \(\, \nu_i = \log\left(\dfrac{\mu_i}{1-\mu_i}\right)\)
Log√≠stico multinomial \(y_i \sim \text{Multinomial}(p_1,\dots,p_k)\) \(\nu_j = \beta_{0j} + \beta_{1j} x_1 + \dots + \beta_{pj} x_p\) (para \(j=1,\dots,K\)) Logit: \(\, \log\left(\dfrac{P(Y=j)}{P(Y=k)}\right)=\nu_j\)
Poisson (conteos) \(y_i \sim \text{Poisson}(\lambda)\) \(\nu_i = \beta_0 + \beta_1 x_1 + \dots + \beta_p x_p\) Log: \(\, \log(\lambda)=X\beta\)

 

Otros GLM incluyen respuestas continuas no negativas o tiempos de supervivencia.

Regularizaci√≥n en el modelo lineal generalizado

GLMs regularizados

 

  • La regularizaci√≥n se utiliza para evitar el sobreajuste al entrenar GLM con fines predictivos.

  • Usamos el mismo enfoque que con el modelo lineal: se introduce un t√©rmino de regularizaci√≥n en la funci√≥n de p√©rdida que penaliza la magnitud de los coeficientes estimados.

  • Hay un par√°metro de ajuste \(\lambda\) que controla la cantidad de regularizaci√≥n.

  • El valor √≥ptimo de \(\lambda\) se elige mediante remuestreo (validaci√≥n cruzada).

La funci√≥n de p√©rdida de un GLM regularizado incluye un regularizador \(R\):
\[-l(\beta|x,y)+\lambda R(\beta)\]

Tipos de penalizaci√≥n

 

Como en el caso del modelo lineal, la penalizaci√≥n puede ser \(L_1\), \(L_2\) o una combinaci√≥n de ambas:

  1. LASSO:
    \[R(\beta)=||\beta||_1=\sum_j|\beta_j|\]
  2. Ridge:
    \[R(\beta)=||\beta||_2=\sum_j\beta_j^2\]
  3. Elastic net:
    \[R(\beta)=\alpha ||\beta||_1+ \frac{1}{2}(1-\alpha)||\beta||_2\]
