Técnicas Avanzadas de Predicción: Proyecto Final

 

Objetivo

Desarrollar un modelo predictivo para estimar una variable objetivo (numérica o categórica) a partir de las características de un conjunto de datos elegido por vosotros.

Etapas

  1. Definir el problema y elegir la clase de modelo según la naturaleza de la variable objetivo:
    • Regresión: Si la variable objetivo es numérica (p.ej., precio, cantidad).
    • Clasificación: Si la variable objetivo es categórica (p.ej., clase, categoría).
    • Regresión + clasificación
  2. Preparar los datos
    • Identificar la variable objetivo y las variables predictoras.
    • Gestionar valores faltantes.
    • Codificar variables categóricas (p.ej., one-hot encoding).
    • Normalizar o escalar variables si es necesario.
    • Evitar data leakage (no usar información del test en el entrenamiento).
  3. Explorar el dataset
    • Analizar distribuciones y detectar outliers.
    • Revisar correlaciones con la variable objetivo.
    • Visualizar relaciones clave (gráficos, histogramas, etc.).
  4. Selección y entrenamiento del modelo
    • Elegir algoritmos adecuados (p.ej., regresión lineal, árbol de decisión, random forest, redes neuronales).
    • Dividir el dataset en train/test.
    • Ajustar hiperparámetros por validación cruzada.
  5. Evaluación
    • Utilización métricas apropiadas: ECM o R2 para regresión; métricas derivadas de la matriz de confusión o AUC para clasificación.
    • Evaluar el modelo en el conjunto de prueba.
    • Comprobar si hay sobreajuste (comparar train vs test).
  6. Interpretación
    • Analizar importancia de variables o coeficientes.
    • Extraer conclusiones sobre los factores más relevantes.

Algunos aspectos que se valorarán para máxima puntuación

  1. Implementación de técnicas desarrolladas en el curso (p.e. estimadores compuestos/pipelines en lugar de transformaciones individuales)
  2. Tratamiento de datos faltantes (imputación)
  3. Tratamiento de datos con desequilibrio de clases (o desbalanceados) para problemas de clasificación
  4. Interpretabilidad de los resultados mediante valores de Shapley
